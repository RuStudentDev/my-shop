# MyShop

Проект, в котором показан принцип работы с корзиной и остатком товаров

## Описание

- `src/components` Папка с компонентами
- `src/data` Папка с хранилищами
- `src/index.tsx` Главный файл запуска

## Для запуска

```bash
> npm install
> npm run dev
```

## Для сборки

```bash
> npm install
> npm run build
```
